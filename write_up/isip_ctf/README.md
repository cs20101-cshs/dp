# ISIP-CTF write-up 教學示範

本區放置 ISIP-CTF 平台題目的解題紀錄與 write-up 範例。

## write-up 基本格式

每篇 write-up 建議包含：
- 題目資訊
- 解題思路
- 解題方法
- 學習重點（老師版使用，學生版可改成 `學習心得與發現`）
- flag 或解題結果（**學生必寫本欄，但老師版不會寫給你看**）

## 題目資訊

建議包含：
- 題目名稱
- 類型，例如 Crypto、Web、Misc、Forensics
- 使用工具
- 相關觀念

## 關於 flag

- 本區題目以教學題、公開題或練習題為主，學生自己的 write-up 可以寫出 flag 與完整解題過程。
- 老師示範用文件不會直接公開 flag，目的是避免同學只 COPY 衝分，而不實際操作。
- 若題目仍在正式比賽期間，或平台明確禁止公開解法，則不得公開 flag 與完整解法。
