# 因才網「資通安全實務」課程學習路徑索引

本文件為 PDF 對照索引，整理因才網「資通安全實務」單元與高中資安教材模組的對應關係，適合教師查找課程位置與規劃訓練順序。

## 建議使用方式：
- 一般學生先看「資安認知教材」與「資安生活教材」
- 有意投入證照或競賽，並以證明撐得過因才網影片教學者，再看「資安基礎實務教材」與「新_資安基礎實務教材」
- 若要導入 CTF ，可優先參考「新_資安基礎實務教材」中的 CTF 模組
