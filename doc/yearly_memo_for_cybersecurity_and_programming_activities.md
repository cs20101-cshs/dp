# 資安、程式競賽 / 活動學年度行事簡曆暨備忘錄

## 八月
- [課程] AIS3 Junior 課程
    - <https://ais3.org/junior>
    - 六月上旬至七月上旬報名並繳交甄選資料
    - 七月中公告錄取名單，錄取者方可參加
    - 八月安排 4 日實體課程
- [檢定] iPAS 資訊安全工程師中級檢定（年度第二次）
    - <https://ipd.nat.gov.tw/ipas/certification/ISE/news>
    - 八月下旬測驗
- [課程] AIS3 好厲駭
    - <https://ais3.org/AIS3holyhack>
    - 六月至七月報名並繳交甄選資料
    - 八月中旬公布初選通過名單，通過者即為「好厲駭高階培訓學員」
    - 八月底課程開始啟動
- [報名] APCS 十月場次
    - <https://apcs.csie.ntnu.edu.tw/>
    - 八月中旬到九月上旬報名
    - 實作提供「初級」、「中級」
    - 參加十月場次，即不得參加十一月場次
- [報名] APCS 十一月場次
    - <https://apcs.csie.ntnu.edu.tw/>
    - 八月中旬到九月上旬報名
    - 實作提供「中級」、「中高級」、「高級」
    - 參加十一月場次，即不得參加十月場次
- [報名] 資訊月資訊應用技能競賽
    - <https://www.infomonth.com/ICTIntro.aspx>
    - 八月下旬至九月中旬報名
    - 由科主任統一報名，全科視情況報名「程式語言 Python 組」、「商用專業編輯組」（類似學生技藝競賽文書處理職種）等。
    - 程式語言 Python 組可 1~5 人組隊參加，每組分數最高前三名列入團體賽成績。
    - 比賽日期可能會和「金盾獎」衝突，須確實比對該年度報名須知後，視情況擇一參加。

## 九月
- [報名] 資安技能金盾獎
    - <https://csc.nics.nat.gov.tw/>
    - 九月中旬至十月上旬報名
    - 以隊為單位，每隊 1~3 人，可跨校、年級、科系組隊
    - 初賽日期可能會和「資訊月資訊應用技能競賽」衝突，須確實比對該年度報名須知後，視情況擇一參加。
- [報名] 第一學期 Bebras 國際運算思維挑戰賽
    - <https://bebras.csie.ntnu.edu.tw/>
    - 九月下旬到十月下旬報名
    - 由「程式語言與設計」任課教師安排全班報名參賽，所以想參加的話就去拗他
- [升學] 檢閱普通大學特殊選才簡章
    - 九月中旬起開始檢閱各大學特殊選才簡章，以「資安外加名額」為檢閱重點，但部分學校仍可注意資工、資管系一般名額與條件。
