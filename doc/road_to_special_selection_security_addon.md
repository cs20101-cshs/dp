# 特殊選才資安外加名額之路
版本：v1.4

本文件供竹山高中資料處理科教師同仁挑選適合學生接受資安特訓之用，
學生自己看了想嘗試也不反對，反正......（絕大多數只是隨口說說）

## 使用方式
本文件可作為校內挑選資安特訓對象與規劃訓練方向之參考。  
建議先看「注意事項」、「挑人」，確認學生是否值得投入；再依實際能力規劃「證照路線」，最後才決定是否進入「競賽路線」。

## 注意事項
1. 此路線**屬特攻性質**，每屆能穩定拉出 1～2 名已屬難得。  
2. 以本校學生能力與投入程度，基本上走到 iPAS 初級 + CCST 就不錯了，再往上則是個案突破。  
3. 學生三分鐘熱度很正常，啟動後自然放生、半途脫隊實屬常態。  
   

## 挑人

1. 高一數位科技概論 -> 成績很高、投入學習  
   **特別觀察：上冊 Ch 2, 4, 5, 11\~15、下冊 Ch 1\~5、10\~12**
2. 高一程式語言與設計 -> 不排斥寫程式  
   不需要像程式選手那麼會寫，但起碼簡單的能自己寫，最好不要只會 key-in
3. 對資訊安全特訓有投入熱情才啟動
4. 高一下開始就能啟動訓練，以證照路線為主，競賽路線視學生個別能力再啟動

## 證照路線

1. 基本目標：iPAS 資訊安全工程師初級（TQC 等雜照就免了）  
2. 進階目標：iPAS 資訊安全工程師中級；取得後，基本上可申請大多數科技大學與部分地名大學資安相關特殊選才名額  
3. 附加目標：Cisco CCST 三張入門級證照（Networking、Cybersecurity、IT Support）  
   - Cisco 好歹是網路大廠，證照還算有說服力  
   - 證照難度不算高，基本難度約略等同熟讀數位科技概論，進階題目可搭配 iPAS 一起準備，至不濟也有考古題或考照資料  
   - 牌價每張 4200，很貴，要問代理商（碁峰）看有沒有學生優惠價  
4. 考照準備詳見  
   [iPAS資訊安全工程師準備指南](./ipas_security_learning_guide.md)
   

## 競賽路線

1. MyFirstCTF，每年四月報名，五月舉行，單人競賽  
   [https://ais3.org/mfctf/](https://ais3.org/mfctf/)  
2. AIS3，每年四月報名，五月 pre-exam，通過了才錄取，七月舉行  
   [https://ais3.org/](https://ais3.org/)  
3. AIS3 沒選上，還可以報名甄選 AIS3 Junior，每年六～七月報名，八月舉行  
   [https://ais3.org/junior](https://ais3.org/junior)  
4. 資安技能金盾獎，每年九月報名，十月初賽，1\~3 人一隊  
   [https://csc.nics.nat.gov.tw/index.aspx](https://csc.nics.nat.gov.tw/index.aspx)  
5. AIS3 EOF CTF，每年十一～十二月報名，十二月初賽，4 人一隊  
   [https://ais3.org/eof/](https://ais3.org/eof/)  
6. 其他 CTF 或資安素養競賽，麻煩資媒組或科主任多注意

>**特別提醒**
> - MyFirstCTF、資安技能金盾獎、AIS3 EOF CTF 得獎，或完整參加 AIS3 正規培訓者，可直接特殊選才所有資安外加名額，且優先權高於 iPAS 中級。
> - 不適合全面推動，主要作為少數高潛力學生之加速與升級管道。技術能力有限的學生走證照路線＋其他低門檻活動補經歷即可

## 競賽訓練

1. 因才網資安課程有教 CTF，可作為基礎訓練，詳見  
   [因才網「資通安全實務」課程學習路徑索引](./it_security_course_in_adl.md)  
2. 申請國家高速網路與計算中心雲端資安攻防平臺（簡稱[CDX](https://cdx.nchc.org.tw/)）作為學生練習平臺，避免干擾學校教學與行政網路運作。  
3. 從高中資安教學資源與推廣中心（ISIP）的 [CTF靶機](https://ctf.isip-hs.org/)開始訓練，上手後打 [picoCTF](https://picoctf.org/)（純英文，但可利用 AI 翻譯），練到一定程度再投入相關競賽。
   [ISIP CTF 平台註冊教學](./isip_ctf_intro.md)
   5. 要求學生**開設 GitHub**，記錄學習筆記與解題思路，除可輔助學習，尚可供營隊甄選乃至於特殊選才備審資料之用。
   [GitHub 開設教學](./simplest_github_intro.md)
   6. 走 CTF 路線的選手仍需要培養基本的程式能力，因此行有餘力可兼練 Python 程式設計，並鼓勵報考 **APCS**，以**識讀 3 級分 / 實作 2 級分**為基本目標。若當年度無正規程式設計選手，尚可跨界參賽。
   [程式設計自學指南](./programming_self_training_guide.md)

## 特殊選才資安外加名額共通資格
  
所有校系一律採認：
> 1. 數發部資安署**資安技能金盾獎**得獎  
> 2. 參加 **AIS3** 取得結業證書  
> 3. 教育部 **MyFirstCTF、AIS3 EOF CTF** 得獎

其他採認資格列表請詳閱該年度簡章彙整
- [一般大學](https://srecruit.moe.edu.tw/)
- [科技大學](https://www.jctv.ntut.edu.tw/enter42/s42/)

各校其他採認資格（如 iPAS 初級／中級、APCS 級分、GiCS、其他競賽或證照）仍有差異，請以當年度正式簡章為準。
