# 程式設計自我鍛鍊與競賽入門指南
版本：v1.1

本文件提供竹山高中資料處理科學生自我鍛鍊程式設計能力參考，也適合有意報考 APCS、參與程式設計競賽，或走資安路線補強程式能力的同學使用。

## 使用方式
1. 不要一開始就把所有連結全部點開。
2. 請依照「一切的起點 → 基本蹲馬步 → 真正開始練功 → 進階路線」的順序往下走。
3. 若只是想補程式能力，不一定要把進階路線全部跑完。

## 入坑前自我檢核
**請先確認自己符合下列三點：**
  1. 確定高一「程式語言與設計」老師指派作業都是自己寫，而且寫得出來。
  2. 不排斥寫程式的歷程（包括寫不出來的卡關和無盡的 debug），甚至有一點點喜歡。
  3. 願意投入自己的休閒時間練習寫程式或學習資安，且有心成為選手。

## 設定目標
1. **透過特殊選才管道進入中字輩（包含台科大、北科大）以上學校**
    - 老師自己也做不到，因為老師小時候沒有多元管道，只有大學聯考，成績要顧
    - 這要 APCS 實作 4 以上 / 總和 7~8 以上
    - 請把下列內容跑到完，或許還有一絲絲可能
2. **走資安路線補程式能力**
    - 老師對各位的預設值
    - 請盡量練到 APCS 識讀 3 / 實作 2（拉到 4/3 更佳）
    - 「進階路線」可不練
3. **單純想當商業類科學生技藝競賽─程式設計職種選手**
    - 技術上要練到目標 2 的程度，但不一定要去考 APCS，然後找老師要考古題去刷

## 一切的起點
- [APCS 官網](https://apcs.csie.ntnu.edu.tw/)
    - 整個網站好好逛一下，本檢測已經成為大學資字輩參採的極重要依據
    - 雖然一年開很多場，但不是每場都會開設所有級別，請考量自己的行程和練習進度報考

- [安裝 APCS 檢測系統 / 練習環境](https://apcs.csie.ntnu.edu.tw/index.php/environment/)
    - 請拋開以往慣用的程式開發環境，起碼在訓練 APCS 期間都要強迫自己用這一套，以免真正去檢定的時候發現自己不會用

- [APCS Code::Blocks 操作教學](https://hackmd.io/@smallshawn95/apcs_codeblocks)
    - 老師沒在用這套，不過你用得習慣的話，好歹這是檢測系統允許的開發工具
    - 平常被 VS Code、Spyder 慣壞的同學請一定要改；真正去檢定時，環境不會因為你平常習慣 VS Code 就特別體貼你

## 基本蹲馬步
1. **「程式語言與設計」課本習題全刷過**
    - 理論上你來找老師之前，這條件的進度應該已經有 70% 以上
2. [師大資工國際運算思維挑戰賽（Bebras，海狸）](https://bebras.csie.ntnu.edu.tw/)
    - 每學年比兩次，兩次題目相同（！）
    - 不用自己報，開資科程式設計課的老師理論上會全班一起測
    - 想吃資工 / 資管這口飯，PR 最好要有 70；如果只有 50 ，但下定決心，還可以靠勤能補拙
3. [陳子晴，Python 入門](https://medium.com/ccclub/all?topic=python-%E5%85%A5%E9%96%80)
    - 課本讀不慣或是臨時忘記基本語法可用
    - 別懷疑，老師就時常忘記 f string 怎麼用
4. [資訊文生研習室](https://itvincent.blogspot.com/)
    - 點選右側邊攔 TQC+ → 程式語言 Python 3
    - TQC+ 的 Python 3 版題庫詳解，TQC+ 的題目是精熟基本語法用的，不用浪費錢去考這張，反正他們辦的程式比賽不看證照......

## 真正開始練功
1. [**ZeroJudge 高中生程式解題系統**](https://zerojudge.tw/)
    - 最常用，也最多人用的線上程式評分系統
    - 缺乏自律的同學可以註冊後找老師「開課」，盯你刷題進度
2. [**中正資工吳邦一教授 PythApcs123 綜合包**](https://drive.google.com/drive/folders/1mnVdO2LHq7e4vesn6pt_R0-S6YWtz4Q4)
    - **必練！！！！！**
    - 內含 Python 教材文件、程式識讀 125 題
3. [PythApcs123 Youtube 播放清單](https://www.youtube.com/playlist?list=PLpmg1QLbgMuSIDOgOcwf0Fbbn2ZDR7s-X)
    - 不想用讀的可以看影片，但吳教授的解說頗有催眠效果......
4. [**APCS 實作題 1級-3級 練習題清單**](https://docs.google.com/spreadsheets/d/1m3PsJACKsJfFxSsRz2xE5zsaqTc6iBwp6l2Ml0tokBc/edit?gid=1468689738#gid=1468689738)
    - 漸進練功刷題清單，已停止更新，用的就是 ZeroJudge 題目
5. [iTSA 程式自學平臺](https://etutor2.itsa.org.tw/)
    - 號稱是「教育部智慧創新關鍵人才躍升計畫」的程式自學平臺，老師沒用過 XDDDDD
